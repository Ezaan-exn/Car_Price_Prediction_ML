# 🚗 Car Price Prediction (ML Project)

## 📌 Overview
This project is a Machine Learning web app that predicts the price of a used car based on user inputs.

The model is trained using real-world data and deployed using Streamlit.

---

## 🎯 Features
- Predict car price in Lakhs
- Simple and user-friendly interface
- Real-time prediction
- Clean and optimized model

---

## 🧠 Machine Learning Model
- Algorithm: Random Forest Regressor
- Accuracy: ~75% (R² Score)
- Features used:
  - Kilometers Driven
  - Seats
  - Mileage
  - Car Age
  - Fuel Type
  - Transmission
  - Owner Type

---

## ⚙️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
